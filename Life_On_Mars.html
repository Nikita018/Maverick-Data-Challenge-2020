def random_colours(number_of_colors):
    colors = []
    for i in range(number_of_colors):
        colors.append("#"+''.join([random.choice('0123456789ABCDEF') for j in range(6)]))

Story in a nutshell

  1. There were 20% null values present in the dataset for one of the variable 'Shadow_In_Midday'. The values have been replaced with the mean of all the values of the column
  2. All variables have numerical values but on studying the variables, we see that Neighbourhood type and turf are actually categorical. Using one-hot encoding to represent these variables.
  3. There is class imbalance in the data. 75% of the data is for 2 kinds of plants.
  4. There are no outliers detected. In most cases, 25% of points are way higher than the 75% mark and thus maybe part of the distribution, so they are not being considered outliers.
  5. Three classification models are compared for this multiclass classification problem - Naive Bayes Classifier, Random Forst and K-Nearest Neighbour
  6. The metric used for comparison of the prediction by the models is weighted F1 score
